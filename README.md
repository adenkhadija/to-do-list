📝 To-Do List App

A simple and interactive to-do list application built using HTML, CSS, and JavaScript. This app allows users to add, complete, and delete tasks in a clean and minimal UI.

Features: 
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

----
Tech Stack:
- HTML – Structure
- CSS – Styling & layout
- Functionality & DOM manipulation


----
Upcoming Features: 
- Save tasks using localStorage (persist after refresh)
- Edit existing tasks
- Add due dates or deadlines
- Filter tasks (Completed / Incomplete)
- Dark mode toggle
- Improved mobile responsiveness
